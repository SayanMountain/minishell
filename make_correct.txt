builtin:
- echo -n (печатает в новой строке)
- cd открывать папки

export
- сортировать по названию


cd и другие команды с флагами
- флаги должны игнорировать любое количество пробелов, например:
    cd -          (1 пробел)
    cd   -        (2 пробела)
    cd          - (10 пробелов)


exit
- команда exit не выходит из bash, если идет с пайпами
    exit | pwd
    ls | exit


после логики прикруить underscore  _=/usr/bin/env
